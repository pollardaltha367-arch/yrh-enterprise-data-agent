## Publish Commands

```bash
git init
git add .
git commit -m "Initial enterprise data analysis workflow"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/yrh-enterprise-data-agent.git
git push -u origin main
```

Do not commit `_external_references/`. It is ignored by `.gitignore`.
