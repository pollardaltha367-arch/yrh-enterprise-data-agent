# Resume Description

## Chinese

中小企业经营数据分析与 AI Agent 工作流：基于零售/电商业务场景，构建订单、商品、用户、库存、投放、评论等数据表，使用 Python/Pandas 完成数据质量检查、经营指标计算和业务分析，输出 GMV、净销售额、客单价、退款率、毛利率、渠道 ROI、库存风险等指标，并自动生成 HTML 看板与 Markdown 经营分析报告。项目同时沉淀数据理解、数据质量、业务分析和报告生成 Skill，为后续 AI 问数和企业数据分析 Agent 扩展打基础。

## English

Enterprise Data Analysis Agent Workflow: Built a lightweight retail/e-commerce analytics pipeline with Python and Pandas. The project loads or generates order, product, user, inventory, marketing, and review data; performs data quality profiling, KPI calculation, sales/category/channel/user/inventory analysis; and automatically generates a static HTML dashboard and Markdown business report. The project also includes reusable skill drafts for future AI-powered data analysis agents.
